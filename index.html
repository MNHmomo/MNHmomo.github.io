a TL;DR News Section

Nearly 12,000 API keys and passwords found in AI training dataset

Truffle Security found nearly 12,000 valid API keys and passwords in the Common Crawl dataset, where companies like OpenAI, Google, and Meta train their AI models. AWS root keys and MailChimp API keys, amongst others, were hardcoded into HTML and JavaScript code and were therefore vulnerable to exploitation. The keys were reused on thousands of pages, with one WalkScore API key being utilized over 57,000 times. Truffle Security helped vendors to revoke affected keys, highlighting the risks of insecure coding on AI training data.

What is a VPN?

Essentially, a virtual private network, or VPN, is a secure private network that is established by encrypting an immense network. A  VPN keeps hackers and attackers, or at least tries its best to, from accessing your computer systems and private information by enabling you to browse anonymously without being discovered by unauthorized users.
