About()

BashBreach is a blog, an indie web site, and a place for referencing my cybersecurity topics, research, projects, and more. I’ll usually document the things I’m learning, researching, and building. You'll see blog entries discussing a range of cybersecurity topics, the latest security tools and trends, the best learning resources I use, and projects I’ve built and how I’ve built them. I try to go in depth with the blog entries, to really make sure I explain the topics well. And if a blog ends up containing far too much content, I always include a TL;DR (Too Long; Didn’t Read) section at the beginning, to summarize the main points. There is also a news section in the home page that I tend to update every morning with the latest cybersecurity news. It features a short, summarized version of the news and a link to the article.

I created this website because I believed if I researched and wrote about cybersecurity, I'd understand its concepts a lot better.This website isn’t just a reference for me, but also for you —especially if you’re a local student or just someone interested in cybersecurity.
